# 🚀 CalcSci - Calculatrice Scientifique Moderne

Une calculatrice scientifique élégante construite en une seule page avec Tailwind CSS et JavaScript.

## Fonctionnalités principales
- Calculs basiques et scientifiques (sin, cos, tan, log, ln, √, etc.)
- Mode Degrés / Radians
- Historique des calculs
- Support complet du clavier
- UI glassmorphism moderne et responsive

## Utilisation
1. Clone le repo ou télécharge `index.html`
2. Ouvre le fichier dans ton navigateur
